# 🎨 SGAC - Interface (Front-end)

Aplicação Web modular e responsiva para o **Sistema de Gestão de Atividades Complementares (SGAC)**. Esta interface foi projetada para oferecer uma experiência fluida para os alunos enviarem seus certificados e para os coordenadores validarem as horas.

## 🚀 Tecnologias Utilizadas
* HTML5 Semântico & CSS3
* JavaScript (ES6 Modules)
* **PWA (Progressive Web App):** Suporte a Service Workers para cache e navegação otimizada.
* **Hospedagem:** GitHub Pages

## 📦 Funcionalidades Principais
* **Sistema de Login Modular:** Redirecionamento dinâmico baseado no perfil de acesso (Aluno, Coordenador, Super Admin).
* **Gestão de Estado:** Controle seguro de sessão e navegação de páginas (`state.js`).
* **Comunicação Assíncrona:** Integração completa com a API RESTful via `fetch`, com tratamento de erros e Toasts de notificação.
* **Cache Inteligente:** Arquivos e assets armazenados via Service Worker para carregamento instantâneo.
